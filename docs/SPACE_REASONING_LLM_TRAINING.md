# Space Reasoning LLM Training Plan

## 🎯 Mission

Train LLaMA 3 8B to **reason about space physics** using mathematical principles, not just mimic NASA documents.

**Goal**: LLM that solves novel orbital mechanics problems with step-by-step work.

---

## 🔄 The Pivot

### **Old Approach** (Abandoned)
- Train on NASA mission reports
- Result: Writes like NASA, but no reasoning
- Problem: Pattern matching, not understanding

### **New Approach** (Current)
- Train on **reasoning traces**: Problems → Solutions with math steps
- Result: Can solve novel problems, explain physics
- Key: Show the thinking process, not just outputs

---

## 📊 Training Data Strategy

### **Synthetic Generation** (Primary: 5,000 examples)

**Format per example**:
```json
{
  "messages": [
    {
      "role": "user",
      "content": "Calculate delta-v for Hohmann transfer from 6771 km to 42164 km"
    },
    {
      "role": "assistant", 
      "content": "**Step 1: Current velocity**\nv₁ = √(μ/r₁)...\n[full solution with math]"
    }
  ]
}
```

**Topics** (generated by `space_reasoning_generator.py`):
1. Hohmann transfers (30%)
2. Orbital periods (25%)
3. Escape velocity (20%)
4. Inclination changes (25%)

**Why synthetic**: Scale to 10K examples, perfect format, no copyright issues.

### **Validation Set** (500 real textbook problems)
- Curtis "Orbital Mechanics" problems
- MIT OCW problem sets
- Purpose: Test generalization beyond synthetic data

## 🚨 Common Issues & Fixes

### **Loss not decreasing**
- ✅ Increase learning rate (1e-4 → 2e-4)
- ✅ Check data format (verify JSONL correct)
- ✅ Increase LoRA rank (32 → 64)

### **OOM (Out of Memory)**
- ✅ Reduce batch size (2 → 1)
- ✅ Increase gradient accumulation (8 → 16)
- ✅ Enable gradient checkpointing

### **Generated text is gibberish**
- ✅ Check tokenizer padding
- ✅ Verify max_length settings
- ✅ Try lower temperature (0.7 → 0.3)

### **Model memorizes training data**
- ✅ Increase dropout (0.05 → 0.1)
- ✅ Add weight decay
- ✅ Reduce epochs (5 → 3)
